PRE-COLLISION  PRE-COLLISION SYSTEM  DETAILS  PRE-COLLISION SYSTEM CANCEL SWITCH  


OPERATION


a.

A pre-collision system cancel switch is provided to disable the system. When pressed and held for approximately 3 seconds, the pre-collision system will be disabled and the PCS warning light will be illuminated. When the switch is pressed again while the system is disabled, the system will be reenabled.


b.

The switch is also used to change the warning timing in 3 levels: Early, Middle, Later. The warning timing changes in the order below each time the switch is pressed.
