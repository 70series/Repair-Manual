SETUP  WHEN REPLACING OR REMOVING/INSTALLING PARTS  UPDATE ECU SECURITY KEY  


UPDATE ECU SECURITY KEY


NOTICE:
·

It is necessary to log in to the Toyota server to perform an ECU security key update.

·

Users other than Toyota dealers can log in with a guest account issued when an ECU is ordered.



a.

Enter the following menus: System Select / Health Check.


b.
2.198,0.438 2.438,0.292 true 0.177,0.167 0.49,0.365 0.313,0.198 10 false *a 2.042,0.385 2.354,0.583 0.313,0.198 10 false *b
*a GTS Health Check Screen
*b "ECU Security Key: Necessary"


Check whether "ECU Security Key: Necessary" is displayed in the upper right corner of the GTS screen.


c.
1.906,1.927 2.208,1.792 true 0.156,0.146 0.469,0.344 0.313,0.198 10 false *a 1.75,1.865 2.063,2.063 0.313,0.198 10 false *b
*a GTS Health Check Screen
*b Perform ECU Security Key Update Icon


If "ECU Security Key: Necessary" is displayed, follow the prompts on the GTS screen to update the ECU security key.


d.

When the ECU security key update is completed, enter the following menus: System Select / Health Check.


e.
2.198,0.427 2.438,0.281 true 0.177,0.167 0.49,0.365 0.313,0.198 10 false *a 2.042,0.385 2.354,0.583 0.313,0.198 10 false *b
*a GTS Health Check Screen
*b "ECU Security Key: -"


Check that "ECU Security Key: -" is displayed, and clear the RoB.

NOTICE:

If "ECU Security Key: Necessary" is displayed in the upper right corner of the GTS screen, perform the ECU security key update again.
