1GR-FE ENGINE CONTROL  SFI SYSTEM  P2610  ECM / PCM Internal Engine Off Timer Performance  


DESCRIPTION

The soak timer is built into the ECM. While the engine is running, the ECM monitors the synchronization of the soak timer and the CPU clock. If these two are not synchronized, the ECM will interpret this as a malfunction, illuminates the MIL and store this DTC.

CAUTION / NOTICE / HINT

HINT:
·

DTC P2610 is set if an internal ECM problem is detected. Diagnostic procedures are not required. ECM replacement is required.

·

Read freeze frame data using the GTS. The ECM records vehicle and driving condition information as freeze frame data the moment a DTC is stored. When troubleshooting, freeze frame data can help determine if the vehicle was moving or stationary, if the engine was warmed up or not, if the air fuel ratio was lean or rich, and other data from the time the malfunction occurred.
