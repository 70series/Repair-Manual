PRE-COLLISION  PRE-COLLISION SYSTEM  FREEZE FRAME DATA  


DESCRIPTION


a.

When a pre-collision system DTC is stored, the forward recognition camera stores the current vehicle (ECU or sensor) state as Freeze Frame Data.



CHECK FREEZE FRAME DATA


a.

Select a DTC to display the freeze frame data.

Body Electrical > Pre-Collision System > Trouble Codes
50001 432

b.

Check the freeze frame data for the output DTC.
