DESCRIPTION

The skid control ECU (brake actuator assembly) outputs this DTC when it receives an internal malfunction signal from the steering sensor.

DTC No. Detection Item DTC Detection Condition Trouble Area
C052696 Steering Angle Sensor Module Component Internal Failure With the +BS terminal voltage 9.5 to 17.4 V, the skid control ECU receives an error signal from the steering sensor. Steering sensor (Spiral cable with sensor sub-assembly)

PROCEDURE

1.REPLACE SPIRAL CABLE WITH SENSOR SUB-ASSEMBLY
a.

Replace the steering sensor (spiral cable with sensor sub-assembly).
