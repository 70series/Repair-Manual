BRAKE CONTROL / DYNAMIC CONTROL SYSTEMS  VEHICLE STABILITY CONTROL SYSTEM(w/ Pre-collision System)  TEST MODE PROCEDURE  


ACTIVATE DEALER MODE (SIGNAL CHECK MODE)


HINT:
·

VSC does not operate in dealer mode (signal check mode), regardless of whether the sensor check is normal or abnormal.

·

During dealer mode (signal check mode), the VSC system does not operate regardless of whether the system is normal or a malfunction is detected.

·

When a malfunction occurs in an ABS system sensor, the ABS warning light illuminates.

·

When a malfunction occurs in a VSC system sensor, the slip indicator light illuminates.

·

When entering dealer mode (signal check mode), the skid control ECU communicates with the forward recognition camera to judge whether the vehicle is equipped with the pre-collision system. Therefore, do not exit dealer mode (signal check mode) within 5 seconds of entering dealer mode (signal check mode).



a.

Procedure to enter dealer mode (signal check mode).

i.

Turn the ignition switch off.


ii.

Check that the shift lever is in neutral.


iii.

Apply the parking brake.


iv.

Put the steering wheel in the neutral position.


v.

Turn the ignition switch to ON.


vi.

Switch the skid control ECU to Dealer Mode (Signal Check) using the GTS.

Chassis > Brake > Utility

HINT:

Do not exit dealer mode (signal check mode) within 5 seconds of entering dealer mode (signal check mode). If dealer mode (signal check mode) is exited within 5 seconds of entering dealer mode (signal check mode), DTC U023A87 may be stored. If DTC U023A87 is output after exiting dealer mode (signal check mode), enter dealer mode (signal check mode) again, wait at least 5 seconds, then recheck for DTCs.




b.

Confirm the system has entered dealer mode (signal check mode)

i.

Confirm that the ABS warning light and slip indicator light indicate the dealer mode (signal check mode) pattern (blinking at 0.25-second intervals).

HINT:

If the ABS warning light or slip indicator light remains illuminated or does not illuminate when dealer mode (signal check mode) is entered, a component or wire harness may be malfunctioning.

ACCELERATION SENSOR CHECK


a.

Enter dealer mode (signal check mode).


b.

Keep the vehicle stationary on a level surface for 1 second or more.

HINT:

The acceleration sensor check can be performed with the master cylinder pressure sensor check below.




MASTER CYLINDER PRESSURE SENSOR CHECK


a.

With the vehicle stopped, release the brake pedal for 1 second or more and check that value of dealer mode (signal check mode) Data List item Output Master Cylinder Pressure Sensor changes from incomplete to complete.

HINT:
·

The status of the ABS warning light and slip indicator light will not change.

·

If the master cylinder pressure sensor is normal and the above step is performed when in dealer mode (signal check mode), the master cylinder pressure sensor check will complete normally.





SPEED SENSOR CHECK


NOTICE:

Before performing the speed sensor signal check, complete the acceleration sensor and master cylinder pressure sensor checks.


a.

Enter dealer mode (signal check mode).


b.

Start the engine.


c.

Drive the vehicle straight ahead.

Accelerate the vehicle to a speed of 45 km/h (28 mph) or more for several seconds and check that the ABS warning light goes off.

·

The sensor check may not be completed if wheelspin occurs.

·

The ABS warning light blinks when the sensor check has been completed and the brake pedal is depressed.

·

The ABS warning light comes on immediately after a malfunction has been detected during the speed sensor check.


HINT:

If the sensor check completes successfully, the ABS warning light turns off when the vehicle is being driven, and indicates the dealer mode (signal check mode) pattern (blinking at 0.25-second intervals) when the vehicle is stopped.




END OF SENSOR CHECK


a.

If the sensor check is completed, the ABS warning light and slip indicator light blink (Test Mode (signal check)) when the vehicle stops and the ABS warning light is off while the vehicle is being driven.

NOTICE:
·

When dealer mode (signal check mode) is started with the GTS and all items display complete, the sensor check ends.

·

If the sensor checks have not completed, the ABS warning light will remain blinking while the vehicle is being driven, and the ABS will not operate.

·

If a DTC is detected in dealer mode (signal check mode), the ABS warning light or slip indicator light illuminates.

·

If any DTCs are output or if dealer mode (signal check mode) items do not complete even after repeated attempts, repair or replace the malfunctioning part.
