BRAKE CONTROL / DYNAMIC CONTROL SYSTEMS  VEHICLE STABILITY CONTROL SYSTEM(w/ Pre-collision System)  C124000  Yaw Rate Sensor / Multi-axis Acceleration Sensor Module "A"  


DESCRIPTION

The airbag sensor with ECU assembly has a built-in yaw rate and acceleration sensor.

If the identification ID of the yaw rate and acceleration sensor (airbag ECU assembly) is different to that stored in the skid control ECU (brake actuator assembly), this DTC is stored.

DTC No. Detection Item DTC Detection Condition Trouble Area
C124000 Yaw Rate Sensor / Multi-axis Acceleration Sensor Module "A" The yaw rate and acceleration sensor (airbag ECU assembly) identification information stored in the skid control ECU (brake actuator assembly) is not consistent with the yaw rate and acceleration sensor (airbag ECU assembly) identification information sent via CAN communication continuously for 1 second or more.
路

Incorrect yaw rate and acceleration sensor (airbag ECU assembly) installed

路

Skid control ECU (brake actuator assembly)
