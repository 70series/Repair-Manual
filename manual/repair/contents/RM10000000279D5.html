PRE-COLLISION  FRONT CAMERA SYSTEM  FREEZE FRAME DATA  


DESCRIPTION


a.

When a front camera system DTC is stored, the forward recognition camera stores the current vehicle (ECU or sensor) state as Freeze Frame Data.



CHECK FREEZE FRAME DATA


a.

Select a DTC to display the freeze frame data.

Chassis > Front Recognition Camera > Trouble Codes
50001 430

b.

Check the freeze frame data for the output DTC.
