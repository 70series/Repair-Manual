THEFT DETERRENT / KEYLESS ENTRY  IMMOBILISER SYSTEM  B2784  Antenna Coil Open / Short  


DESCRIPTION

When an open or short circuit is detected in the transponder key amplifier coil built into the transponder key ECU assembly, the transponder key ECU assembly stores this DTC.

DTC No. Detection Item DTC Detection Condition Trouble Area Note
B2784 Antenna Coil Open / Short The antenna coil in the transponder key coil is open/shorted (1 trip detection logic*1).
·

Harness or connector

·

Transponder key coil

·

Transponder key ECU assembly
DTC Output Confirmation Operation:

Insert the key into the ignition key cylinder.

*1: Only output while a malfunction is present.

Vehicle Condition when Malfunction Detected Fail-safe Operation when Malfunction Detected
