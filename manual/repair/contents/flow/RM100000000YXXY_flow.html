·

Inspect the fuses of circuits related to this system before performing the following procedure.

·

After replacing the ECM, the new ECM needs registration (Click here Engine / Hybrid System>1VD-FTV (ENGINE CONTROL)>ECD SYSTEM (w/ DPF)>REGISTRATION 201608,999999,_51,_029760,_0282525,RM100000000Z1DB, ) and initialization (Click here Engine / Hybrid System>1VD-FTV (ENGINE CONTROL)>ECD SYSTEM (w/ DPF)>INITIALIZATION 201608,999999,_51,_029760,_0282525,RM100000000YXU9, ).

·

Read freeze frame data using the GTS. Freeze frame data records the engine condition when malfunctions are detected. When troubleshooting, freeze frame data can help determine if the vehicle was moving or stationary, if the engine was warmed up or not, and other data from the time the malfunction occurred.
