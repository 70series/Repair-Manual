DESCRIPTION

The skid control ECU (brake actuator assembly) receives signals from the yaw rate sensor and acceleration sensor (airbag ECU assembly) via CAN communication.

HINT:

If there is a malfunction in the bus lines between the yaw rate sensor and acceleration sensor (airbag ECU assembly) and the CAN communication system, DTCs U012587 are output. When DTCs U012587 are output together with C00632A, C05201C, C05202A and/or C052096, inspect and repair the trouble areas indicated by U012587 first.


DTCs may be stored if either of the following occurs:

·

Acceleration sensor (airbag ECU assembly) installation abnormality.

·

Acceleration sensor (airbag ECU assembly) signal malfunction.
