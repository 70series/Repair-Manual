THEFT DETERRENT / KEYLESS ENTRY  IMMOBILISER SYSTEM  B2796  No Communication in Immobiliser System  


DESCRIPTION

This DTC is stored when a key is inserted into the ignition key cylinder but no communication occurs between the key and transponder key ECU assembly.

DTC No. Detection Item DTC Detection Condition Trouble Area Note
B2796 No Communication in Immobiliser System The key ID code cannot be transmitted to the transponder key ECU assembly (1 trip detection logic*).
·

Key

·

Transponder key coil

·

Transponder key ECU assembly
DTC Output Confirmation Operation:

Insert the key into the ignition key cylinder.

*: Only output while a malfunction is present.
