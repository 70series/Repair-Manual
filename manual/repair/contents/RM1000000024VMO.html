PRE-COLLISION  PRE-COLLISION SYSTEM  U041881  Invalid Data Received from Brake System Control Module Invalid Serial Data Received  


DESCRIPTION

·

The forward recognition camera receives signals from the skid control ECU (brake actuator assembly) corresponding to the brake control status via the CAN communication line.

·

When a brake control system malfunction signal is received from the skid control ECU (brake actuator assembly), the forward recognition camera outputs U041881.


DTC No. Detection Item DTC Detection Condition Trouble Area
U041881 Invalid Data Received from Brake System Control Module Invalid Serial Data Received Once 3 seconds elapse after turning the ignition switch to ON, the forward recognition camera receives a brake control system malfunction signal from the skid control ECU (brake actuator assembly) for approximately 1 second or more.
·

Forward recognition camera

·

Vehicle stability control system

CAUTION / NOTICE / HINT

NOTICE:
·

When replacing the forward recognition camera, always replace it with a new one. If a forward recognition camera which was installed to another vehicle is used, the information stored in the forward recognition camera will not match the information from the vehicle. As a result, a DTC may be stored.

·

When the forward recognition camera is replaced with a new one, make sure to read Before Starting Adjustment, then perform optical axis alignment for the forward recognition camera and clear the vehicle control history (RoB) for each system.

HINT:

Forward recognition camera axis alignment can be performed by using either One Time Recognition, Sequential Recognition or Driving Adjustment.
