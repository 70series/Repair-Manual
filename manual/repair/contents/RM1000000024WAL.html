BRAKE CONTROL / DYNAMIC CONTROL SYSTEMS  VEHICLE STABILITY CONTROL SYSTEM(w/ Pre-collision System)  C124A00  ECM Communication  


DESCRIPTION

The circuit is used to send TRC and VSC control information from the skid control ECU (brake actuator assembly) to the ECM, and engine control information from the ECM to the skid control ECU (brake actuator assembly) through the CAN communication system.

DTC No. Detection Item DTC Detection Condition Trouble Area
C124A00 ECM Communication Either of the following is detected:
1.

Engine model information sent from the ECM does not match the information stored in the skid control ECU.

2.

No system information stored in the skid control ECU.

3.

Incorrect installation of the skid control ECU or ECM.

4.

When the skid control ECU is malfunctioning.
·

ECM malfunction or ECM with incorrect part number installed

·

Skid control ECU (Brake actuator assembly)
