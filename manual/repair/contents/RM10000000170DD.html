THEFT DETERRENT / KEYLESS ENTRY  IMMOBILISER SYSTEM  B2795  Unmatched Key Code  


DESCRIPTION

This DTC is stored when a key with a different vehicle serial number is inserted into the ignition key cylinder.

DTC No. Detection Item DTC Detection Condition Trouble Area Note
B2795 Unmatched Key Code A key with a different vehicle serial number is inserted into the ignition key cylinder (1 trip detection logic*).
路

Key

路

Transponder key ECU assembly
DTC Output Confirmation Operation:

Insert the key into the ignition key cylinder.

*: Only output while a malfunction is present.

Vehicle Condition when Malfunction Detected Fail-safe Operation when Malfunction Detected
