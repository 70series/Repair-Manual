BRAKE CONTROL / DYNAMIC CONTROL SYSTEMS  VEHICLE STABILITY CONTROL SYSTEM(w/ Pre-collision System)  C006328  Yaw Rate Sensor Signal Bias Level Out of Range / Zero Adjustment Failure  C006396  Yaw Rate Sensor Component Internal Failure  


DESCRIPTION

The skid control ECU (brake actuator assembly) receives signals from the yaw rate and acceleration sensor (airbag ECU assembly) via the CAN communication system.

The airbag ECU assembly has a built-in yaw rate and acceleration sensor and detects the vehicle condition.

If there is trouble in the bus lines between the acceleration sensor (airbag ECU assembly) and the CAN communication system, DTCs U012587 are set.

These DTCs are also output when calibration has not been completed.
