BRAKE CONTROL / DYNAMIC CONTROL SYSTEMS  VEHICLE STABILITY CONTROL SYSTEM(w/ Pre-collision System)  C051D54  Multi-axis Acceleration Sensor Module "A" Missing Calibration  C121054  Yaw Rate Sensor Missing Calibration  


DESCRIPTION

The skid control ECU (brake actuator assembly) receives signals from the yaw rate and acceleration sensor (airbag ECU assembly) via the CAN communication system.

The airbag ECU assembly has a built-in yaw rate and acceleration sensor and detects the vehicle condition. If there is trouble in the bus lines between the yaw rate and acceleration sensor (airbag ECU assembly) and the CAN communication system, the DTC U012587 is output.

The DTCs are also output when the calibration has not been completed.
