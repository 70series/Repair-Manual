Use YOKOGUSHI

When using YOKOGUSHI, a diagnostic code can be input to search for related information from the Repair Manual, New Car Features, and Electrical Wiring Diagram.

Input the diagnostic code and click on YOKOGUSHI.
YOKOGUSHI results are displayed.

HINT
  • YOKOGUSHI can be searched using a five-digit diagnostic code. Two-digit diagnostic codes cannot be used for searches.
  • Only alphabet characters, numerical characters and the symbol " / " can be used for the diagnostic code.

Click on the link for the item you want to check.
Relevant contents are displayed.

How to Read Search Results

Results for keyword searches and YOKOGUSHI are displayed in the Results (search results) tab. Click on the link for the item you want to check, to display the corresponding content.

A

Search result label
For a keyword search, “Results For “***”” is displayed (where *** is the keyword).
For YOKOGUSHI, “YOKOGUSHI Results For "***”” is displayed (where *** is the diagnostic code).
B Manual label
This displays whether there is a search result in each manual.
It is displayed in blue if there is one or more search hits, and in white if there are no hits.
C Search results
The number of search results in each manual, and the search result items, are displayed.
Items are displayed in the sequence Categories, Sections, Titles, Contents.
